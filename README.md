# [JAVA-ARC] ICP Assignments Answers for first year.

This repository contains solutions to ICP assignment questions. Each file represents the answer to a specific question provided in the assignments.

p.s. if there are any mistakes in any of the answers please do submit a pull request!

## File Structure

The project follows the structure below:

```
AssignmentX (e.g., Assignment6)
├── JRE System Library [JavaSE-1.8]
├── src
│   ├── (default package)
│   │   ├── HAQ1.java      # Home Assignment Question 1
│   │   ├── HAQ2.java      # Home Assignment Question 2
│   │   ├── Q1.java        # Question 1
│   │   ├── Q2.java        # Question 2
│   │   ├── Q3.java        # Question 3
│   │   ├── Q4.java        # Question 4
│   │   ├── Q5.java        # Question 5
│   │   ├── Q6.java        # Question 6
│   │   ├── Q7.java        # Question 7
│   │   ├── Q8.java        # Question 8
│   │   ├── Q9.java        # Question 9
│   │   ├── Q10.java       # Question 10
├── Lab Assignment X.pdf    # Assignment description file
```

## How to Use

1. Clone the repository:
   ```
   git clone https://github.com/mayankkanti/javaarc.git
   ```

2. Open the project in your preferred Java IDE (e.g., Eclipse, IntelliJ IDEA, VSCode, etc).

3. Navigate to the `src` folder and select the desired `.java` file to view or execute the solution.

4. Ensure you have JDK 1.8 or higher installed.
